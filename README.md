# FirstExpressHW

## Installation 
Fork and clone the repository
```
$ git clone https://github.com/yourusername/repositoryname
```
Go to the project directory
```
$ cd repositoryname
```

Install express package
```
$ npm init -y
```
```
$ npm i express
```
Install dotenv package
```
$ npm i dotenv
```
To run the application 
```
$ nodemon
```
